| 指标        | 公式                          | 解释与用途                       |
| :-------- | :-------------------------- | :-------------------------- |
| **夏普比率**  | (组合收益 - 无风险利率) / 组合标准差      | 衡量每单位**总风险**所获得的超额回报。       |
| **特雷诺比率** | (组合收益 - 无风险利率) / 组合Beta     | 衡量每单位**系统风险**所获得的超额回报。      |
| **信息比率**  | (组合收益 - 基准收益) / 跟踪误差        | 衡量每单位**主动风险**所获得的额外回报。      |
| **索提诺比率** | (组合收益 - 最低要求收益) / 下行标准差     | 衡量每单位**下行风险**所获得的超额回报。      |
| **跟踪误差**  | σ(组合收益 - 基准收益)              | 衡量投资组合收益相对于基准收益的波动程度。       |
| **詹森阿尔法** | 组合收益 - [Rf + β*(市场收益 - Rf)] | 衡量经市场风险调整后，超越基准的超额收益。       |
| **M²测度**  | Rf + (组合夏普比率 * 市场标准差)       | 将组合风险调整至市场风险水平后，与市场直接比较的收益。 |

- **业绩评估逻辑**
- **被动投资**追求低跟踪误差，紧密复制基准。
- **主动投资**追求高信息比率，意味着用更少的主动风险冒险获得了更高的主动回报。

### **核心概念解析**

1. **风险调整的核心不同，应用场景不同**
- **夏普比率**调整的是**总风险（波动率）**, 评估**未完全分散**的投资组合。
- **特雷诺比率**调整的是**系统风险（Beta）**, 评估**已充分分散**的投资组合。
- **信息比率**调整的是**主动风险（跟踪误差）**, 考核**主动型基金经理**的能力。
- **索提诺比率**调整的是**下行风险**, 投资者对**下方损失特别敏感**时。

2. **下行标准差计算**
- 确定最低可接受收益率。
- 仅收集收益率低于MAR的数据点。
- 计算这些数据点与MAR的偏离程度的平方的平均数，然后开方。

3. **关键误区提醒**
- **信息比率**的分母是跟踪误差（标准差），不是简单的收益差。
- **索提诺比率**的分母是**下行标准差**，不是全部波动率，也不是“好”的波动率。
- **跟踪误差**衡量的是波动性，而不是收益的高低。
