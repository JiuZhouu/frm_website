### 1. 基本类型

| 类型 | 英文 | 核心特征 |
| :--- | :--- | :--- |
| **固定缴款型计划** | **Defined Contribution Plan** | 雇主/雇员按约定金额或比例缴费，退休福利取决于账户的投资收益。 |
| **固定收益型计划** | **Defined Benefit Plan** | 雇主承诺按特定公式支付退休福利，风险主要由雇主承担。 |

### 2. 核心特征对比

| 方面 | 固定缴款型 | 固定收益型 |
| :--- | :--- | :--- |
| **风险承担** | **雇员** 承担投资风险 | **雇主** 承担投资风险、长寿风险 |
| **承诺内容** | 承诺**缴款**金额 | 承诺**退休福利**支付额 |
| **账户形式** | 个人账户 | 养老金资金池（非个人独立账户） |
| **投资决策** | 通常由**雇员**选择投资选项 | 由**雇主/管理人**负责资产配置 |
| **负债管理** | **无** 需计算养老金负债 | 需精算计算**养老金义务**，并进行资产-负债管理 |
| **资金状况** | 总是**完全足额**（因账户即资产） | 可能存在**资金缺口** |
| **雇主义务** | 按时按约定缴费 | 确保资金充足，足以支付未来所有债务 |
| **政府担保** | 通常**无** | 通常**有**（如美国的PBGC） |

### 4. 风险管理视角

#### **4.1 固定收益型计划的风险**
1.  **投资风险**：资产回报低于负债增长。
2.  **利率风险**：折现率下降导致负债现值上升。
3.  **长寿风险**：参与者寿命超过预期，支付期延长。
4.  **流动性风险**：需要支付养老金时资产无法及时变现。

#### **4.2 固定缴款型计划的风险**
1.  **投资风险**：由雇员承担，可能因投资不当导致退休储蓄不足。
2.  **通胀风险**：投资收益未能超越通货膨胀。
3.  **长寿风险**：个人账户余额可能过早耗尽。

### 5. 会计与监管

- **固定收益型**：
  - 必须在资产负债表上报告养老金资金状况。
  - 需定期进行**精算估值**来确定成本和负债。
  - 资金缺口会增加公司负债，影响信用评级。

- **固定缴款型**：
  - 雇主缴费直接计入利润表作为费用。
  - 资产负债表上不体现养老金负债（因无承诺）。

### 6. FRM考试要点

- 能够清晰区分**固定收益型**与**固定缴款型**的核心区别。
- 理解风险承担主体的不同是根本区别。
- 知道**固定收益型**计划存在**资金缺口风险**，是公司信用分析的重要考量。
- 了解**长寿风险**、**利率风险**对固定收益型计划的影响。
- 熟悉**PBGC**等担保机制的作用和局限。
