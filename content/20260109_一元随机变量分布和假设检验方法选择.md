### 1. 一元随机变量分布

#### 1.1 离散型分布

| **项目**          | **伯努利分布 (Bernoulli)**    | **二项分布 (Binomial)**                                | **泊松分布 (Poisson)**                                          |
| --------------- | ------------------------ | -------------------------------------------------- | ----------------------------------------------------------- |
| **定义**          | 单次试验，成功概率 $P$            | $n$ 次独立伯努利试验的成功次数 $X$                              | 单位时间内随机事件发生次数，适用于 $n$ 大 $P$ 小                               |
| **公式**          | $P(X=1)=P$, $P(X=0)=1-P$ | $P(X=k) = C(n,k) \cdot$<br>$P^k \cdot (1-P)^{n-k}$ | $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$，$\lambda = nP$ |
| **均值 $E[X]$**   | $P$                      | $nP$                                               | $\lambda$                                                   |
| **方差 $Var(X)$** | $P(1-P)$                 | $nP(1-P)$                                          | $\lambda$                                                   |
| **FRM应用**       | 单次违约/事件发生                | 一系列投资的成功次数、违约次数                                    | 操作风险/信用风险事件计数（例如一年内违约次数）                                    |

#### 1.2 连续型分布

| **项目**          | **连续均匀分布 (Uniform)**               | **正态分布 (Normal) N(μ,σ2)**                                             | **对数正态分布 (Lognormal)**            |
| --------------- | ---------------------------------- | --------------------------------------------------------------------- | --------------------------------- |
| **定义**          | 在区间 $[A,B]$ 内等概率取值                 | 对称分布，两端无限延伸                                                           | $Y = \ln X \sim N(\mu, \sigma^2)$ |
| **PDF $f(x)$**  | $f(x) = \frac{1}{B-A}，x \in [A,B]$ | $f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ | -                                 |
| **均值 $E[X]$**   | $\frac{A+B}{2}$                    | $\mu$                                                                 | -                                 |
| **方差 $Var(X)$** | $\frac{(B-A)^2}{12}$               | $\sigma^2$                                                            | -                                 |
| **性质要点**        | **CDF**：$F(x) = \frac{x-A}{B-A}$   | **偏度**：0，**峰度**：3                                                     | $X > 0$；**右偏**；用于股票价格建模           |
| **关键分位点**       | -                                  | $Z_{0.025} = 1.96,\ Z_{0.05} = 1.65,\ Z_{0.01} = 2.33$                | -                                 |

### 2. 抽样分布与统计检验分布

这些分布是构建假设检验的基础，都来源于正态分布。

| **项目**  | **卡方分布 (χ2)**                         | **t 分布 (t)**                                 | **F 分布 (F)**                                            |
| ------- | ------------------------------------- | -------------------------------------------- | ------------------------------------------------------- |
| **定义**  | $k$ 个独立标准正态变量平方和：$\sum_{i=1}^k Z_i^2$ | 标准正态除以卡方随机变量：$T = \frac{Z}{\sqrt{\chi^2/k}}$ | 两个卡方随机变量的比值：$F = \frac{\chi_1^2 / k_1}{\chi_2^2 / k_2}$ |
| **自由度** | $k$                                   | $k$                                          | $k_1, k_2$                                              |
| **均值**  | $k$                                   | 0                                            | -                                                       |
| **方差**  | $2k$                                  | $\frac{k}{k-2}（k>2）$                         | -                                                       |
| **特点**  | 非负、右偏；用于检验**方差**                      | 尾部厚于正态分布；适用于**小样本均值**检验                      | 非负、右偏；用于比较**两总体方差**                                     |

#### 2.1 分布之间的转换

| **分布**                      | **转换条件**                         | **含义**                               |
| --------------------------- | -------------------------------- | ------------------------------------ |
| **二项 $\rightarrow$ 正态**     | $nP \geq 10,\ n(1-P) \geq 10$    | 当试验次数足够大时，可以用正态分布近似二项分布。             |
| **泊松 $\rightarrow$ 正态**     | $\lambda \geq 1000$              | 当事件发生率（$\lambda$）足够大时，可以用正态分布近似泊松分布。 |
| **二项 $\rightarrow$ 泊松**     | $n$ 大，$P$ 小，$\lambda = nP$ 固定    | 用于罕见事件的计数。                           |
| **$t$ 分布 $\rightarrow$ 正态** | $k \rightarrow \infty$ (自由度趋向无穷) | $t$ 分布的自由度越大（样本量越大），越接近标准正态分布。       |

### 3. 假设检验方法的选择

核心是理解检验目的、样本大小和总体方差信息。

| **检验名称**                 | **目的（检验对象）**                    | **样本要求/条件**                                  | **检验统计量**       | **FRM应用场景**             |
| ------------------------ | ------------------------------- | -------------------------------------------- | --------------- | ----------------------- |
| **$Z$ 检验**               | 检验**均值**                        | 总体方差 $\sigma^2$ **已知** 或 样本量 $n \ge 30$（大样本） | **$Z$ 分布**      | 检验大样本下资产组合的平均收益率。       |
| **$t$ 检验**               | 检验**均值**                        | 总体方差 $\sigma^2$ **未知**，且通常用于**小样本** $n < 30$ | **$t$ 分布**      | 检验小样本下，收益率是否显著不为零。      |
| **卡方检验 ($\chi^2$ Test)** | 检验**单个总体方差** $\sigma^2$ 是否等于某常数 | 假设总体服从**正态分布**。                              | **$\chi^2$ 分布** | 检验样本波动性是否符合预期（如风险模型校验）。 |
| **$F$ 检验**               | 比较**两组数据的总体方差**是否相等             | 假设两总体都服从**正态分布**。                            | **$F$ 分布**      | 比较两个资产或策略的波动性差异。        |

#### 💡 记住

- **均值检验**：看**总体方差**是否已知来区分 $Z$ 和 $t$。

- **方差检验**：检验**一个方差**用 $\chi^2$；检验**两个方差**用 $F$。
