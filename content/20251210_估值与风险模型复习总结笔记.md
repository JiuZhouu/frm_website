## 债券基本性质与估值
### 1. 债券 - 估值模型
- **债券估值原理**
	- **债券现金流**：包括利息（coupon）和本金（principal）。
	- **估值方法**：将未来现金流贴现至现值（PV），即：$$P = \sum_{t=1}^{T} \frac{CF_t}{(1 + r_t)^t}$$
	- 其中 $r_t$ 可为即期利率（spot rate）、远期利率（forward rate）或YTM。
- **即期利率（Spot Rate）**
	- **定义**：从零时刻开始到某未来时刻的年化利率。
	- **特征**：
		- 年化利率。
		- 所有复利计算需考虑时间调整（如半年期需除以2）。
		- 即期利率与到期时间的关系是利率期限结构（term structure）。
- **远期利率（Forward Rate）**
	- **定义**：从未来某一时刻到更远未来时刻的利率。
	- 所有远期利率可通过即期利率推导得出。
	- 无套利定价原则：$$(1 + S_1)^{T_1} \times (1 + F_{T_1T_2})^{T_2 - T_1} = (1 + S_2)^{T_2}$$
	- 连续复利形式为：$$S_1T_1 + F_{T_1T_2}(T_2 - T_1) = S_2T_2$$
- **利率期限结构**
	- **类型**：
		- Upward sloping（上升型）：长期利率高于短期。
		- Downward sloping（下降型）：短期利率高于长期。
		- Flat（平坦型）：各期限利率相同。
	- **应用**：用于预测经济趋势、制定投资策略。
- **利率趋平的期限结构 A flattening term structure occurs**
	- **牛市趋平**：长期和短期利率均下跌，但长期利率下跌幅度大于短期利率；
	- **熊市趋平**：长期和短期利率均上涨，但短期利率上涨幅度大于长期利率。
- **利率陡峭化的期限结构 A steepening term structure occurs**
	- 长期和短期利率均下跌，但短期利率下跌幅度大于长期利率；
	- 长期和短期利率均上涨，但短期利率上涨幅度小于长期利率。
- **折现因子（Discount Factor）**
	- **定义**：未来一单位货币在当前的价值。
	- **公式**：$$D_t = \frac{1}{(1 + \frac{S_t}{m})^{m \cdot t}}$$
	- 其中 $m$ 为年付息次数。
	- 已知折现因子可反推即期利率：$$S_t = \left(\frac{1}{D_t}\right)^{1/(m \cdot t)} - 1$$
	- 应用
		- 简化债券定价计算（将除法转换为乘法）。
		- 例：债券现金流为 $C_1, C_2, ..., C_n$ $，则定价为：$$P = C_1D_{t_1} + C_2D_{t_2} + \cdots + C_nD_{t_n}$
- **平价收益率（Par Rate）**
	- **定义**：债券价格等于面值（FV）时的票面利率（coupon rate）。
	- **计算公式**（以半年付息为例）：$$\text{Par Rate} = \frac{1 - D_T}{\sum_{i=1}^{n} D_{t_i}} \times 2$$
	- 其中 $D_T$ 为最后一期折现因子，$D_{t_i}$ 为各期折现因子。
	- **意义**：用于债券市场报价与定价校准。
- **债券定价方法比较**  
	- 即期利率法：$$P = \frac{C_1}{(1 + \frac{S_1}{2})^1} + \frac{C_2}{(1 + \frac{S_2}{2})^2} + \cdots$$
	- 折现因子法：$$P = C_1D_{0.5} + C_2D_1 + C_3D_{1.5} + C_4D_2$$远期利率法：
		- 分阶段折现，先折至中期再折至零时刻。
	- **优劣对比：**
		- 即期利率法：需多次幂运算，较复杂。
		- 折现因子法：简化计算，适合考试。
		- 远期利率法：用于理解现金流路径，适合模型推导。
- **净价与全价** 
	- **Clean Price（净价）**：不含应计利息的债券价格；用于市场报价。
	- **Dirty Price（全价）**：含应计利息的债券价格；实际成交价格。
	- **dirty price = clean price + accrued interest**
	- **应计利息（Accrued Interest）**：
		- 定义：自上次付息日至交易日之间应计的利息。
		- 计算方式：$$AI = \frac{\text{实际持有天数}}{360} \times {C}$$
- **一价定律与复制法**：
	- **一价定律**：相同现金流应有相同价格；若偏离，则存在套利机会。
	- **复制法**：用零息债券组合复制附息债券现金流。
- **年金** 
	- **年金定义**：一系列方向相同、金额相等、时间间隔一致的现金流。
	- 与债券区别：无本金偿还。
	- **普通年金公式**：$$P = \frac{C}{Y} \left(1 - \frac{1}{(1 + \frac{Y}{2})^{2T}}\right)$$
		- 其中 $Y$ 为YTM，$T$ 为年数。
	- **永续年金（perpetuity bond）公式**：$$P = \frac{C}{Y}$$
		- 适用于无限期现金流。
		- **历史背景**：起源于英国在拿破仑战争中发行的“永续债券”，用于筹措军费。
### 2. 债券 - 收益率
- **实际收益率（Gross Realized Return）**
	- 公式为：$$R = \frac{P_{t+1} + C - P_t}{P_t}$$
	- 其中 $P_t$ 为买入价格，$P_{t+1}$ 为卖出价格，$C$ 为期间收到的利息。
- **净收益率（Net Realized Return）**
	- **与Gross的区别**：净收益率考虑了融资成本，即买入债券的资金是否来自借款。
	- **公式调整**：若资金来自借款，需将借款本息计入成本：$$R_{\text{net}} = \frac{P_{t+1} + C - B_{funded price}}{P_t}=\frac{P_{t+1} + C - P_t(1 + r_{\text{fin}})}{P_t}$$
	- 其中 $r_{\text{fin}}$ 为融资利率。
	- **分母保持不变**
	- 不论是否考虑融资成本，分母始终为初始买入价格 $P_t$。
- **到期收益率YTM（Yield to Maturity）**
	- YTM 是使债券未来现金流的现值等于当前价格的唯一折现率。
	- **计算方法**：以现金流折现法为基础，通过公式或计算器求解：$$P = \sum_{t=1}^{n} \frac{C}{(1+Y)^t} + \frac{FV}{(1+Y)^n}$$
	- 其中 $C$ 为每期利息，$FV$ 为面值，$Y$ 为 YTM。
	- **YTM 的两个前提条件**
		- 所有现金流以 YTM 利率再投资。
		- 债券持有至到期；
	- **YTM 与即期利率的关系**
		- YTM 是一系列即期利率（Spot Rate）的加权平均；
		- 权重为各期现金流的现值占比；
		- 若期限结构为**向上倾斜（Upward Sloping）**，YTM < 最长期限的 Spot Rate；
		- 若期限结构为**向下倾斜（Downward Sloping）**，YTM > 最长期限的 Spot Rate。
		- 若期限结构为**平坦型（Flat ）**，YTM = Spot。
- **即期利率（Spot Rate）与远期利率（Forward Rate）**  
	- 即期利率：从当前（0时刻）到未来某时刻的利率。
	- 远期利率定义：从未来某时刻到更远未来时刻的利率。
	- **两者关系公式**
	- 以连续复利为例：$$(1 + S_1T_1)(1 + f_{12}(T_2 - T_1)) = (1 + S_2T_2)$$
	- 可推导出：$$f_{12} = \frac{S_2T_2 - S_1T_1}{T_2 - T_1}$$
- **票息效应（Coupon Effect）**
	- 票息效应：票息率变化对 YTM 的影响。
	- **核心逻辑**
		- 高票息债券的现金流更早回收，暴露于利率风险的时间更短；
		- 因此，高票息债券的 YTM 更受短期利率影响。
	- **结论**
		- 在 Upward Sloping 时，YTM 随票息上升而下降；
		- 在 Downward Sloping 时，YTM 随票息上升而上升。
- **债券定价与利差（Spread）**
	- **债券定价方法**
		- 未来现金流折现法；
		- 使用 Spot Rate、Forward Rate 或 YTM 进行贴现。
	- **Spread 的定义**
		- Spread = 债券收益率 - 无风险利率（如国债收益率）；
		- 反映信用风险、流动性风险等。
	- **Spread 的应用**
		- 用于公司债与国债的定价差异分析；
		- 可作为风险补偿的衡量。
- **损益构成（Profit and Loss Components）**
	- **损益构成的两个部分 Decomposition of P&L**
		- Cash Carry：期间收到的利息；
		- Capital Gain/Loss：买卖价差。
	- **Capital Gain/Loss 的进一步分解**
		- **Carry Roll Down**：仅因时间流逝导致的价格变动 + 利息收入；
		- **Rate Change**：市场利率变动导致的价格变动；
		- **Spread Change**：利差变动导致的价格变动。
### 3. 债券市场 - 风险 - 平行移动
- **债券定价基础**：债券价格等于未来现金流贴现求和。
- **债券市场风险的来源**
	- 市场风险（Market Risk）：主要由利率变动引起的价格波动。
	- 信用风险（Credit Risk）：主要出现在公司债中，指发行人违约的可能性。
- **利率水平**
	- 是债券价格变动的唯一变量，未来现金流已知，因此利率变动直接影响债券市场价格。
- **利率期限结构**
	- 利率期限结构是指不同期限的利率水平，通常以时间（年）为横坐标、利率水平为纵坐标。
	- 向上倾斜（Upward Sloping）
	- 向下倾斜（Downward Sloping）
	- **单因素假设  One-Factor Assumption**：假设利率改变受单一因素影响。
	- **平行移动（Parallel Shift）**
		- 定义：利率期限结构整体平移，形状不变。
		- 举例：一年期、五年期、十年期利率均上升相同数值（如从3%、5%、8%变为4%、6%、9%）。
		- 特点：所有期限利率变动量相同（Δy相同）。
	- **非平行移动（Non-Parallel Shift）**
		- 定义：不同期限利率变动量不同。
		- 特点：无法直接通过单一利率变动推导所有期限利率变化。
- **麦考林久期（Macaulay Duration）**
	- **定义**：以折现现金流为权重的平均现金流回流时间。
	- 公式：$$D_{\text{Macaulay}} = \sum_{t=1}^{n} \left( \frac{PV(CF_t)}{P} \times t \right)$$
	- 对于普通债券，麦考利久期小于或等于其到期日。 
	- 对于零息债券，麦考利久期等于其到期日。
- **修正久期（Modified Duration）**
	- 定义：衡量利率变动一个单位时，债券价格变动的百分比。
	- 公式：$$\text{Modified Duration} = \frac{D_{\text{Macaulay}}}{1 + \frac{y}{m}}$$
	- 其中 $y$ 为年利率，$m$ 为每年付息次数。
- **麦考林久期与修正久期的关系**
	- **麦考林久期 vs 修正久期**
		- 麦考林久期：以时间加权现金流的平均回流时间。
		- 修正久期：麦考林久期除以 $1 + \frac{y}{m}$，用于衡量利率变动对价格的百分比影响。
	- **连续复利情形下的关系**
		- 在连续复利情况下，麦考林久期等于修正久期。
- **美元久期（Dollar Duration）**
	- 定义：利率变动一个单位时，债券价格变动的绝对金额。
	- 公式：$$\text{Dollar Duration} = D_{\text{Modified}} \times P$$
- **DV01（Dollar Value of 01）**
	- 定义：利率变动一个基点（0.01%）时，债券价格变动的金额。
	- 公式：$$\text{DV01} = D_{\text{Modified}} \times P \times 0.0001$$
- **久期在对冲中的应用**
	- **完全对冲（Perfect Hedge）**
		- 定义：通过另一个债券对冲，使组合的利率风险为零。
		- 公式：$$F_B = -\frac{DV01_A}{DV01_B} \times F_A$$
		- 其中 $F_A$、$F_B$ 为债券A、B的面值，负号表示反向操作（如A为多头，则B为空头）。
	- **对冲比率（Hedge Ratio）**
		- 定义：为对冲某一债券，所需对冲债券的数量。
		- 公式同上，基于DV01计算。
- **凸性（Convexity）及其作用** 
	- **久期的限制**：对非平行移动或大幅度的平行移动，效果不好
	- **凸性**：衡量债券价格对利率变动的非线性关系。
- **修正凸性（Modified Convexity）**
	- 公式：$$C_{\text{Modified}} = \frac{1}{(1 + \frac{y}{m})^2} \sum_{t=1}^{n} \left( \frac{PV(CF_t)}{P} \times t(t + 1) \right)$$
- **价格变动的完整公式（含久期与凸性）**
	- $$\Delta P \approx -D_{\text{Modified}} \times P \times \Delta y + \frac{1}{2} \times C_{\text{Modified}} \times P \times (\Delta y)^2$$
- **凸性的经济意义**
	- 正凸性意味着：利率下降时价格上涨更多，利率上升时价格下跌更少。
	- 凸性越大，对债券投资者越有利。
- **久期与凸性的组合应用**
	- **组合久期的计算**
		- 组合久期为各资产久期的加权平均，权重为各资产价格占组合总价格的比例。$$D_P = \sum w_i \times D_i$$
	- **组合凸性的计算**
		- 类似久期，组合凸性为各资产凸性的加权平均。$$C_P = \sum w_i \times C_i$$
	- **子弹型（Bullet）与杠铃型（Barbell）组合对比**
		- 子弹型：资产集中在中等期限。
		- 杠铃型：资产集中在短期和长期。
		- 在利率期限结构平行移动时，杠铃型组合因凸性更高，通常表现更优。
- **有效久期与有效凸性（适用于含权债券）**
	- **有效久期（Effective Duration）**
	- 定义：利率上下变动相同基点后，债券价格变动的平均百分比。
	- 公式：$$D_{\text{Effective}} = \frac{P^- - P^+}{2 \times P_0 \times \Delta y}$$
	- **有效凸性（Effective Convexity）**
	- 定义：利率上下变动后，债券价格变动的非线性部分。
	- 公式：$$C_{\text{Effective}} = \frac{P^- + P^+ - 2P_0}{P_0 \times (\Delta y)^2}$$
	- **适用范围**
	- 有效久期与有效凸性主要用于含权债券（如可赎回债券、可回售债券），因其现金流不确定性，传统久期与凸性不再适用。
- **零息债券的久期与凸性特性** 
	- **麦考林久期**
		- 零息债券的麦考林久期等于其到期期限。
	- **修正久期**
		- 修正久期为麦考林久期除以 $1 + \frac{y}{m}$ 。
	- **凸性计算**
		- 公式：$$C = \frac{T(T + 1)}{(1 + \frac{y}{m})^2}$$
		- 其中 $T$ 为到期年数。
- **利率变动下的价格估算**
	- **仅用久期估算价格变动**$$\Delta P \approx -D_{\text{Modified}} \times P \times \Delta y$$
	- **结合久期与凸性估算价格变动**$$\Delta P \approx -D_{\text{Modified}} \times P \times \Delta y + \frac{1}{2} \times C_{\text{Modified}} \times P \times (\Delta y)^2$$
	- **精确计算 vs 估算对比**
		- 修正久期估算存在误差，加入凸性后估算更接近真实价格变动。
- **含权债券的久期与凸性特征** 
	- **可赎回债券（Callable Bond）**
		- 表现：在利率下降时，发行人可能赎回债券，导致债券价格上升受限。
		- 凸性：表现为负凸性。
	- **可回售债券（Puttable Bond）**
		- 表现：在利率上升时，持有人可将债券回售给发行人，价格下跌受限。
		- 凸性：表现为正凸性。
	- **利率互换对久期的影响**
		- 收浮动付固定：久期为正，利率上升时价格上升。
		- 收固定付浮动：久期为负，利率上升时价格下降。
- **久期与凸性的综合应用** 
	- **对冲策略：久期中性**
		- 目标：使组合久期为零，避免利率变动带来的价格波动。
		- 方法：利用久期相反的债券进行对冲。
	- **对冲策略：久期与凸性双中性**
		- 目标：使组合久期与凸性均为零，实现更稳健的对冲。
		- 方法：建立两个对冲资产，解两个方程，分别使久期与凸性为零。
- **重点**
	- **平行移动下的久期分析**
		- 麦考林久期：现金流的平均回流时间。
		- 修正久期：衡量利率变动对价格变动的百分比。
		- 美元久期：衡量利率变动对价格变动的绝对值。
		- DV01：利率变动1个基点时的价格变动。
	- **凸性分析**
		- 凸性越大，价格对利率变动的响应越有利。
		- 凸性是二阶导数，修正久期是一阶导数。
		- 含权债券的久期与凸性
		- 有效久期与有效凸性适用于含权债券。
		- 可赎回债券具有负凸性，可回售债券具有正凸性。
	- **组合久期与凸性**
		- 组合久期为各资产久期的加权平均。
		- 组合凸性为各资产凸性的加权平均。
		- 子弹型组合适用于非平行移动，杠铃型组合适用于平行移动。
	- **利率互换与久期方向**
		- 收浮动付固定：正久期。
		- 收固定付浮动：负久期。
	- **久期与凸性在对冲中的应用**
		- 久期中性：使组合久期为零。
		- 久期与凸性双中性：使组合久期与凸性均为零。
### 4. 债券市场 - 风险 - 非平行移动
- **主成分分析**适用于提取主要风险因子，简化建模；**关键利率敞口方法**更适用于实际风险对冲与管理。
- **非平行移动** 
	- 利率在不同期限上变动方向或幅度不同，如短期利率下降、长期利率上升，或中间期限反向变动等。
	- 平行移动在现实中几乎不存在，非平行移动是市场风险分析的重点。
- **主成分分析（PCA）**
	- 一种统计方法，用于分析不同期限利率变动的风险因子影响。
	- 通过历史数据提取主要风险因子，衡量其重要性与相关性.
	- **应用前提**：所选因子彼此不相关；利率变动与风险因子之间为线性关系；前两三个因子主要影响变动。
	- **主成分分析示例**
		- 表格数据展示不同期限利率（1年、2年、3年、5年、10年等）对8个风险因子的敏感度。
		- 每个因子载荷（Factor Loading）表示单位因子变动对利率的影响。
		- 因子得分（Factor Scores）衡量每个因子的重要性。
		- 前两个因子可解释约95%的利率变动，前三个因子解释率达97.66%。
	- **核心思想** 
		- 删繁就简，保留主要风险因子，忽略次要影响。
		- 降低维度，便于风险建模与分析。
- **关键利率敞口（Key Rate Exposure）**
	- **基本原理** 
		- 选取若干关键利率（如2年、5年、10年等）作为影响因素。
		- 假设其他利率受关键利率线性递减影响。
	- **关键利率影响范围** 
		- 关键利率仅影响其相邻期限利率。
		- 影响程度按线性递减方式分配，如三角形权重。
		- 左右无相邻关键利率时，影响横向延伸而非归零。
	- **叠加效应**  
		- 若多个关键利率同时变动，其对某一期限利率的影响为各因子影响的叠加。
	- **Key Rate 01**：关键利率变动1BP对债券价格的影响金额。
	- **Key Rate Duration**：关键利率变动对债券价格百分比影响，对标修正久期（Modified Duration）。
	- **Key Rate 01与风险对冲示例** 
		- **组合风险分析**
			- 组合包含1年、3年、5年、9年、15年零息债券。
			- 提供不同期限利率变动1BP时组合价值的下跌金额。
		- **构建Key Rate Shifts矩阵**
			- 选取2年、5年、10年作为关键利率。
			- 计算各关键利率对其他期限利率的影响权重。
			- 构建影响矩阵，用于后续风险敞口计算。
		- **计算Key Rate 01** 
			- 以两年期关键利率为例：
				- 1年期利率变动1BP，影响权重为1，对应价格变动96.72元。
				- 3年期利率变动1BP，影响权重为0.67，对应价格变动270元 × 0.67。
				- 总影响为：1 × 96.72 + 0.67 × 270 ≈ 276.9元。
			- 同理计算5年期与10年期影响。
		- **风险对冲策略**
			- 通过引入对冲工具，使组合Key Rate 01为零。
			- 设定对冲工具的Key Rate 01系数，建立方程组求解所需对冲比例。
			- 负号表示需反向操作（如卖出对冲工具）。
- **Forward Bucket shift（其他方法）**
	- 通过变动远期利率（Forward Rate）计算对债券价格的影响。
	- 每个“桶”代表一段期限内的利率变动。
- **监管要求** 
	- 监管机构通常要求银行使用10个关键利率进行风险分析。
	- 包括：1个月、3个月、6个月、1年、2年、3年、5年、7年、10年、30年。
	- 要求考虑利率标准差与相关系数，计算组合风险标准差，算VaR和ES。
	- **示例计算**  
		- 给定两年期和五年期的Key Rate 01分别为52和97，标准差分别为4和11，相关系数为0.6。
		- 计算组合风险标准差：$$\sigma_P = \sqrt{(52^2 \cdot 4^2) + (97^2 \cdot 11^2) + 2 \cdot 0.6 \cdot 52 \cdot 4 \cdot 97 \cdot 11}$$
		- 结果为：1203.36元
## 期权估值方法
### 1. 二叉树模型
- **二叉树模型**
	- 用于期权基金定价。
	- 可用于美式期权、欧式期权及奇异期权。
	- 模型的使用依赖于硬件与软件支持，可能耗时且成本较高。
- **风险中性世界与投资者行为** 
	- **风险中性（Risk Neutral）**：投资者预期收益率不随风险变化，均为无风险收益率。
	- **风险偏好（Risk Seeking）**：高风险对应低预期收益率，投资者追求刺激。
	- **风险厌恶（Risk Aversion）**  ：高风险对应高预期收益率，投资者要求风险补偿。
- **一步二叉树模型的构造与应用** 
	- 假设：股票价格从20元上涨至22元或下跌至18元，执行价格为21元，无风险收益率为12%，期权为欧式看涨期权。
	- 三个月后：
		- 上涨：看涨期权价值为1元（$S - K = 22 - 21 = 1$）。
		- 下跌：看涨期权价值为0元（$S - K = 18 - 21 < 0$）。
	- 构造无风险组合：买入 $\Delta$ 份股票，卖出1份看涨期权。
	- 通过等式 $22\Delta - 1 = 18\Delta$ 解得 $\Delta = 0.25$。
	- 组合零时刻价值：$20\Delta - F = 5 - F$。
	- 组合一时刻价值：$22\Delta - 1 = 4.5$。
	- 通过无风险折现计算期权价格 $F = 0.633$。

	- **解法2:通过概率计算期权价格** 
	- 设股票价格上升概率为 $P$，下降概率为 $1 - P$。
	- 股票期望值为 $22P + 18(1 - P)$。
	- 根据无风险折现关系：$$20e^{rT} = 22P + 18(1 - P)$$
	- 解得 $P = 0.6523$。
	- 期权期望值为 $1 \times P + 0 \times (1 - P) = P$。
	- 折现得期权价格为 $P \times e^{-rT} = 0.633$。
- **二叉树模型的一般化与公式推导**  
	- $S_0$：当前股价
	- $U$：上涨幅度
	- $D$：下跌幅度
	- $F_U$：股价上涨时的期权价值
	- $F_D$：股价下跌时的期权价值
	- 构造无风险组合：买入 $\Delta$ 份股票，卖出1份看涨期权。
	- 通过等式 $S_0U\Delta - F_U = S_0D\Delta - F_D$ 解得：$$\Delta = \frac{F_U - F_D}{S_0(U - D)}$$
	- 期权价格计算公式：$$F = e^{-rT}[PF_U + (1 - P)F_D]$$
	- 上升概率 $P$ 的计算公式：$$P = \frac{e^{rT} - D}{U - D}$$
	- **U与D的确定方式**
		- 已知数值法：如股价从20元涨至22元，则 $U = 22 / 20 = 1.1$，$D = 18 / 20 = 0.9$。
		- 百分比法：如上涨20%，则 $U = 1.2$，$D = 0.8$。
		- 未知波动率法：假设 $U \cdot D = 1$，$U = e^{\sigma\sqrt{\Delta t}}$，$D = e^{-\sigma\sqrt{\Delta t}}$。
- **多步二叉树模型的应用** 
	- 两步二叉树构造方法：
		- 初始股价为810元。
		- 第一步：股价变为895.19元或732.92元。
		- 第二步：进一步变为989.34元、810元、663.1元。
	- 计算各节点期权价值：
		- 执行价格为800元，欧式看涨期权。
		- 末节点期权价值分别为189.34元、10元、0元。
	- 折现计算期权价格：$$F = e^{-2r\Delta t}[P^2F_{UU} + 2P(1 - P)F_{UD} + (1 - P)^2F_{DD}]$$
	- 计算得期权价格为53.4元。
- **美式期权提前行权判断** 
	- 举例：美式看跌期权，执行价格52元，股价为50元。
	- 末节点期权价值分别为0元、4元、20元。
	- 中间节点折现值为1.4147元和9.4634元。
	- 提前行权价值分别为0元和12元。
	- 若折现值小于提前行权值，则选择提前行权。
	- 最终期权价格为5.09元。
- **不同标的资产的处理方式**  
	- **含红利资产 Options on Stocks with Dividends**：
		- 红利作为 $q$ 被纳入概率计算：$$P = \frac{e^{(r - q)\Delta t} - D}{U - D}$$
	- **指数期权 Options on Stock Indices**：
		- 红利视为 $q$，处理方式同上。
	- **外汇期权 Options on Currencies**：
		- $r$ 为本国利率，$q$ 为外国利率。
	- **期货期权 Options on Futures**：
		- $r = 0$，概率公式简化为：$$P = \frac{1 - D}{U - D}$$
- **二叉树与BSC模型的关系** 
	- 二叉树模型假设股票价格服从离散分布。
	- BSC模型假设股票价格服从几何布朗运动。
	- 随着步数增加，二叉树模型计算结果收敛于BSC模型结果。
### 2. BSM模型
- **BSM模型的特点**  
	-  BSM模型公式简单直接，只需代入参数即可得出期权价值，相比二叉树模型更高效。
	- 优点：计算速度快。
	- 缺点：仅适用于具有欧式期权特征的期权，不适用于美式或奇异期权。
- **BSM模型的发展历程**
	- 1900年：巴切利耶首次研究期权价值，假设股票价格服从正态分布，但因现实不符未能广泛应用。
	- 1965年：萨缪尔森改进模型，假设股票价格服从对数正态分布（log-normal），但仍存在未知参数μ问题。
	- 1973年：布莱克、斯科尔斯、莫顿通过数学推导消去缪，最终推导出BSM模型，模型得以广泛应用。
- **BSM模型的基本假设**
	- **股票价格服从对数正态分布**：
		- 股票价格 $S_t$ 服从 log-normal 分布。
		- 股票收益率 $r$ 服从 normal 分布.
	- **市场为完美市场：**
		- 无交易费用和税费。
		- 所有证券可无限分割。
		- 无红利支付（该假设可放宽）。
		- 不存在无风险套利机会。
	- **无风险利率为常数：**
		- 所有产品收益率均为无风险利率。
	- **连续交易：**
		- 交易连续进行，无时间跳跃。
	- **期权不可提前行权：**
		- 仅适用于欧式期权。
- **股票价格服从对数正态分布 Lognormal stock price**：
	- $$\ln S_t \sim N\left(\ln S_0 + (\mu - \frac{1}{2}\sigma^2)T, \sigma^2 T\right)$$
	- 第一项代表Expected return on stock per year，第二项代表Volotility of the stock per year
	- Since $lnS_T$ is normally distributed, $S_T$ has a lognormal distribution.
	- 股票价格期望值：$$E(S_T) = S_0 e^{\mu T}$$
- **几何布朗运动与随机过程**
	- 股票价格变动模型：$$dS_t = \mu S_t dt + \sigma S_t dz$$
	- 其中：
		- $\mu$：股票收益率（常数）
		- $\sigma$：波动率（常数）
		- $dz$：布朗运动随机项（服从正态分布）
	- 趋势项与波动项：
		- 趋势项：$\mu S_t dt$，代表价格的长期趋势。
		- 波动项：$\sigma S_t dz$，代表价格的随机波动。
- **伊藤引理与期权价值变动**
	- **B-S differential equation**
	- 期权价值变动公式：$$df = \left( \frac{\partial f}{\partial t} + \mu S_t \frac{\partial f}{\partial S} + \frac{1}{2} \sigma^2 S_t^2 \frac{\partial^2 f}{\partial S^2} \right) dt + \sigma S_t \frac{\partial f}{\partial S} dz$$
	- 公式含义：
		- 一阶偏导：期权价值对时间、标的资产价格的敏感度。
		- 二阶偏导：期权价值对标的资产价格变化的非线性影响。
		- 随机项：受标的资产波动率影响。
- **BSM模型公式与参数含义**
	- 欧式看涨期权公式：$$Call = S_0 N(d_1) - K e^{-rT} N(d_2)$$
	- 欧式看跌期权公式：$$Put = K e^{-rT} N(-d_2) - S_0 N(-d_1)$$
	- 参数说明：
		- $S_0$：标的资产初始价格
		- $K$：执行价格
		- $r$：无风险利率
		- $T$：到期时间
		- $\sigma$：标的资产波动率
		- $N(d)$：标准正态分布累计概率
	- $d_1$ 与 $d_2$ 公式：$$d_1 = \frac{\ln(S_0/K) + (r + \frac{1}{2}\sigma^2)T}{\sigma \sqrt{T}}, \quad d_2 = d_1 - \sigma \sqrt{T}$$
	- $N(d_1)$：计算看涨期权的delta（看涨期权对标的资产价格变动敏感性）
	- $N(d_2)$：计算看涨期权的行权概率
	- $1-N(d_2)$：计算看跌期权的行权概率
- **其他期权定价** 
	- **含红利的期权定价 Options on Stocks with Dividends**
		- 处理方式：
			- 对股票价格进行调整：$S_0' = S_0 e^{-qT}$
			- 其中 $q$：红利收益率
	- **货币期权 Options on Currencies**
		- 以外国无风险利率支付股息收益率。
	- **期货期权 Options on Futures**
		- 以国内无风险利率支付股息收益率。
- **隐含波动率（Implied Volatility）**
	- 通过市场期权价格反推得到的波动率，反映市场对未来波动的预期。
	- **计算方式**：已知期权市场价格、$S_0$、$K$、$r$、$T$，反推 $\sigma$
	- **波动率微笑（Volatility Smile）**：隐含波动率随执行价格变化而变化，形成“微笑”曲线。
- **提前行权问题**
	- **美式看涨期权**：
		- 无红利时，不应提前行权。
		- 有红利时，若红利 $D > K(1 - e^{-r(T-t)})$，可考虑提前行权。
	- **美式看跌期权**：
		- 若红利 $D < K(1 - e^{-r(T-t)})$，可考虑提前行权。
- **权证（Warrant）与期权对比**
	- 权证属于一级市场，行权时公司增发新股，稀释现有股东权益。
	- 期权属于二级市场，交易已发行股票，不稀释股权。
	- **权证定价公式**：$$W = \frac{N}{N + M} \cdot C$$
		- $N$：原有股票数量
		- $M$：新增股票数量
		- $C$：对应期权价值
### 3. 希腊字母敏感性分析
- **对冲策略 Hedging Strategy**
	- 对冲是指通过引入对冲头寸来抵消原有风险头寸的风险。
	- **裸头寸 (naked position)**：仅持有期权而无标的资产对冲。
	- **有保护头寸 (covered position)**：期权与标的资产组合形成风险中性组合。
	- **止损策略 (stop-loss strategy)**：通过动态买卖标的资产以维持风险中性，但会导致高买低卖，成本较高。
- **Delta**
	- Delta表示标的资产价格变动一个单位时，期权价格变动的比率。
	- **数学表示：** Delta = ∂ 期权价格 / ∂标的资产价格
	- **图形解释：**Delta是标的资产价格与期权价格曲线的斜率。
	- 看涨期权：Delta ∈ [0, 1]
	- 看跌期权：Delta ∈ [-1, 0]
	- **平值期权 (at the money)**：Delta ≈ 0.5（call），Delta ≈ -0.5（put）
	- **深度实值期权 (deep in the money)**：Delta → 1（call），Delta → -1（put）
	- **深度虚值期权 (deep out of the money)**：Delta → 0
	- **BSM模型中的Delta期权：**
		- 欧式看涨期权：Delta = $N(d_1)$
		- 欧式看跌期权：Delta = $N(d_1) - 1$
- **Delta对冲**
	- **Delta中性 (delta neutral)**：Delta = 0，表示标的资产价格变动不会引起期权价格变动。
	- **对冲方式**：
		- 若Delta > 0：卖出股票（short stock）以对冲。
		- 若Delta < 0：买入股票（long stock）以对冲。
	- **例题分析**：
		- 一个short call的Delta为-0.5812，持有100万份该期权，总Delta为-581200。
		- 对冲方式：买入581200份股票，使Delta = 0。
		- 若Delta变为0.5616，需卖出部分股票以维持Delta中性。
- **Gamma**
	- Gamma表示标的资产价格变动一个单位时，Delta的变动量。对call和put相同。
	- **数学表示：**Gamma = ∂Delta / ∂标的资产价格 = ∂²期权价格 / ∂标的资产价格²
	- **性质：**
		- Gamma > 0：long option（call或put）
		- Gamma < 0：short option（call或put）
		- Gamma最大值出现在at the money附近。
		- Gamma随到期时间缩短而增大
	- **图形解释：**
		- Gamma图形呈钟形曲线，与正态分布相似。
		- 时间越短，Gamma越大；时间越长，Gamma越小。
		- 深度实值或虚值期权：Gamma趋近于0。
- **Gamma对冲策略**  
	- **Gamma中性**：Gamma = 0，表示Delta稳定。
	- **对冲步骤**：
		1. 先使Gamma中性。
		2. 再用股票对冲Delta。
	- 例题分析：
		- 一个组合Gamma为-3000，使用Delta = 0.62、Gamma = 1.5的看涨期权进行对冲。
		- 需要买入2000份期权，使Gamma = 0。
		- 引入Delta = 2000 × 0.62 = 1240，需卖出1240份股票对冲Delta。
- **Vega**  
	- Vega表示波动率变动一个单位时，期权价格的变动量。
	- 实值期权对波动率最敏感。
	- call的vega和put的vega相同。
	- **性质：**
		- Vega > 0：long option（call或put）
		- Vega < 0：short option（call或put）
		- Vega最大值出现在at the money附近
		- 时间越长，Vega越大；时间越短，Vega越小
	- **图形解释：**
		- Vega图形呈钟形曲线，与Gamma图形相似。
		- 时间越长，曲线越宽；时间越短，曲线越窄。
- **Theta**
	- Theta表示时间流逝一个单位时，期权价值的变动量。
	- **性质：**
		- Theta < 0：long option（call或put）
		- Theta > 0：short option（call或put）
		- Theta最大绝对值出现在at the money附近
		- 时间越短，Theta绝对值越大
	- **图形解释**：
		- Theta图形与Gamma图形上下对称。
		- 时间越短，Theta曲线越陡峭。深度实值欧式看跌期权 (deep in the money European put)：Theta可能为正。
- **Delta, Theta和Gamma的关系**
	- 希腊字母与伊藤引理关系，伊藤引理推导BSM模型：$$rF = \frac{\partial F}{\partial t} + rS\frac{\partial F}{\partial S} + \frac{1}{2} \sigma^2 S^2 \frac{\partial^2 F}{\partial S^2}$$
	- 希腊字母对应项：
		- Theta：$\frac{\partial F}{\partial t}$
		- Delta：$\frac{\partial F}{\partial S}$
		- Gamma：$\frac{\partial^2 F}{\partial S^2}$
	- 其他说明：
		- $r$ = the risk-neutral risk-free rate of interest
		- $F$ = the value of the option
		- $\theta$ = the option theta
		- $S$ = the price of the underlying stock
		- $\triangle$ = the option delta
		- $\sigma^2$ = the variance of the underlying stock
- **Rho**
	- Rho表示无风险利率变动一个单位时，期权价格的变动量。
	- **性质**：
		- 看涨期权：Rho > 0
		- 看跌期权：Rho < 0
		- 影响较小：相比其他希腊字母，Rho影响较小。
	- **图形解释：**
		- Rho图形与Theta、Gamma不同，主要受时间影响。
- **投资组合 Portfolio**
	- 依赖于同一资产的衍生品投资组合的任何希腊字母都可以计算为**每个组成部分的希腊字母的加权和**。
	- **Forward Delta**: 1 or $e^{-qT}$
	- **Futures Delta**:  $e^{rT}$ or $e^{(r-q)T}$
	- **组合保险策略 Portfolio Insurance**：通过合成期权（如期货+股票）模拟真实期权的保护效果。
- **希腊字母区分技巧**  
	- **Delta**：标的资产价格变动对期权价格的影响。
	- **Gamma**：标的资产价格变动对Delta的影响。
	- **Vega (V)**：波动率 (Volatility) 变动对期权价格的影响。
	- **Theta (T)**：时间 (Time) 流逝对期权价格的影响。
	- **Rho (R)**：利率 (Rate) 变动对期权价格的影响。
## 市场风险模型
### 1. 金融风险标准
- **一致性风险度量 Coherent Risk Measure**
	- 一致性风险度量是一种标准化的风险测量方法，用于评估某种风险测量方式是否合格。
	- 需要满足四个条件：单调性、次可加性、其次性、位移不变性。
	- **单调性 Monotonicity**：若组合A比组合B可能产生更差的结果，则A的风险更高。
	- **次可加性 Subadditivity**：组合A+B的风险 ≤ A的风险 + B的风险。（VaR通常不满足此条件。）
	- **其次性 Homogeneity**：投资规模放大λ倍，对应风险也放大λ倍。
	- **位移不变性 Translation Invariance** ：加入现金（K）可稀释风险。
- **均值-方差框架  Mean-Variance Framework**
	- 均值代表收益，方差或标准差代表风险。
	- 假设收益服从正态分布。
	- **正态分布的局限性** 
		- 实际金融数据常存在偏态和厚尾。（**skewed or has heavier tails**）
		- 使用均值-方差框架可能导致高估或低估风险。
- **VaR**
	- **VaR的起源** 
		- 由JP Morgan提出，用于一句话总结最大可能损失。
		- How much can we lose on our trading portfolio by tomorrow's close?
		- 一种能全面测量复杂证券组合的市场风险的方法，能够集成不同交易、不同部门的市场风险。
	- VaR is **the maximum loss** over **a target horizon** and for **a given confidence level**.
	- 三个要素：**置信水平、时间区间、最大损失**。
	- 示例：95%置信水平下，一天VaR为40元，表示有95%的概率损失不超过40元。
	- **VaR的两种定义方式**
		- 置信水平下的最大损失。
		- 显著性水平下的最小损失。
	- **VaR的计算示例**
		- 三种损失情境：3 million、6 million、10 million，对应概率88%、8%、4%。
		- 95%置信水平下VaR为6 million。
		- 97%置信水平下VaR为10 million。
		- 96%置信水平下，VaR可选6或10，取决于风险态度。
	- **VaR的局限性** 
		- **不能衡量尾部损失**  
			- VaR仅提供阈值，无法反映超过该阈值后的实际损失。
			- 例如：5%显著性水平下，最小损失为40，但实际损失可能是400、4000甚至更大。
		- **不满足次可加性**
			- 示例：两个贷款资产违约概率均为4%，单个资产VaR为0。
			- 组合后VaR为100，大于单个资产之和，违反次可加性。
		- **其他局限性**
			- 对非正态分布不适用。
			- 高置信水平（如99.99%）下需大量数据支持。
			- 损失量级计算不稳定。
- **ES**
	- ES = Expected Shortfall = 条件风险价值（CVaR）= 尾部损失的平均值。
	- 公式：$\text{ES} = E[L | L > \text{VaR}]$，是连续的。
	- **ES的优势**
		- 能衡量尾部损失。
		- 满足一致性风险度量的四个标准。单调性 ✔ 次可加性 ✔ 其次性 ✔ 位移不变性 ✔
		- 示例：损失为3、6、10，概率为88%、8%、4%，95% ES为9.2。
### 2. VaR
- **市场风险中价值计算的核心方法**：
	- 参数法、非参数法和半参数法的基本原理、优缺点及具体应用场景
	- Delta-Normal法、历史模拟法和平方根法则的计算逻辑及公式推导
- **组合类型与线性关系**
	- 研究对象为投资组合，分为**线性组合与非线性组合**。
	- **线性组合**：标的资产价格变动与衍生品价格变动呈完全线性关系，如远期（forward）和期货（futures）。
	- **非线性组合**：组合中包含期权（option）等衍生品，涉及希腊字母Delta（一阶导）与Gamma（二阶导），呈现非线性特征。
- **参数法与非参数法** 
	- **参数法**：
		- 通过公式计算VaR，通常假设损失分布或收益率服从正态分布。
		- 公式形式：$\text{VaR} = \mu - Z \cdot \sigma$
		- 适用于正态分布假设下的快速计算。
		- VaR用于衡量损失，因此选取置信区间的下限，即单尾检验。
		- **常用Z值**：
			- 95%置信水平：1.65
			- 99%置信水平：2.33
	- **非参数法**：
		- 不依赖公式，基于历史数据排序查找损失值，如历史模拟法（Historical Simulation）。
		- **好处**：
			- 不需要关于整个收益分布的**假设**来估计VaR。
			- 容忍偏态、厚尾等特征。
		- **坏处**：
			- **需要大量样本**才能使用历史模拟精确估计波动性。
			- 将完整的数据样本分为不同的市场体系**减少了历史模拟的可用数据量**。
- **历史模拟法**
	- **步骤：**
		- 搜集历史数据（如100个数据点）
		- 按照损失从大到小排序
		- 根据置信水平查找对应损失值（如95%对应第95或第96个数据）
	- **优点：**
		- 不要求数据服从特定分布
		- 容忍偏态、厚尾等特征
	- **缺点：**
		- 数据量要求大（如至少100个数据）
		- 要求“历史会重演”，即过去数据能代表未来
- **平方根法则与时间调整** 
	- 用于将短期VaR扩展为长期VaR。
	- 公式：$\text{VaR}_T = \text{VaR}_1 \cdot \sqrt{T}$
	- 前提条件：
		- 数据独立同分布（IID）
		- 收益率均值近似为零（适用于短期Horizon）
	- 时间单位转换：
		- 一年通常为250~255个交易日
		- 一周约为5个交易日，一月约为20~22个交易日
- **Delta-Normal法计算衍生品VaR** 
	- 基于一阶导数（Delta）计算衍生品价值变动。
	- 公式：$$\text{VaR}_{\text{option}} = \Delta \cdot \text{VaR}_{\text{underlying}}$$
- **高阶修正与Gamma影响** 
	- 加入Gamma（二阶导）修正项，提升计算精度。
	- 公式：$$\text{VaR}_{\text{option}} = \Delta \cdot \text{VaR}_{\text{underlying}} - \frac{1}{2} \Gamma \cdot (\text{VaR}_{\text{underlying}})^2$$
	- Gamma影响：
		- Long Option：Gamma > 0，风险降低
		- Short Option：Gamma < 0，风险增加
- **蒙特卡罗模拟法** 
	- 通过随机数生成大量情境，模拟组合价值变化。
	- 适用于线性与非线性组合，支持任意分布。
	- 缺点：计算量大，耗时长，成本高。
- **最差情境分析与压力测试**  
	- 补充VaR对尾部风险的忽视。
	- **压力测试方法：**
	- **极端市场情境模拟 Worst Case Analysis**
	- **考虑相关系数上升与波动率**
	- **上升传染效应（感染效应）**： 危机中相关系数与波动率同步上升，导致风险放大
	- A contagion effect
- **相关系数瓦解与金融危机案例** 
	- **Correlation Breakdown**
	- 危机中相关系数显著上升，分散化效果减弱。
	- 次贷危机中低估了结构化产品（如CDO）之间的相关性，导致风险集中爆发。
### 3. 波动率
- **波动率与市场风险的关系** 
	- 波动率是VaR计算中的关键指标，与风险值密切相关。  
	- 公式中波动率（σ）与VaR呈正比关系。  
	- VaR = |μ - Zσ| × Position，其中Z为置信水平对应值，Position为头寸。
- **收益率分布**
	- **收益率分布的非稳定性 Deviations from Normality**  
		- 波动率随时间变化，导致收益率分布不稳定。  
		- 相较于正态分布，实际收益率分布常呈现“肥尾”（fat tail）现象。
	- **条件正态分布与非条件正态分布**  
		- 非条件正态分布：所有时期波动率相同，分布稳定。  
		- **条件正态分布：每个时期波动率不同，但每个时期内部服从正态分布。**
		- 出现 regime switching.
		- **结构调整（structural breaks）会导致波动率突变，增加风险建模难度。**
- **波动率的测量**
	- **收益率定义**  
	- $$R_t = \frac{P_t - P_{t-1}}{P_{t-1}}$$
	- **样本方差公式**  
		- $$s^2 = \frac{1}{M-1} \sum_{i=1}^{M} (R_i - \bar{R})^2$$
	- **简化处理**  
	- 假设收益率均值为0，简化为 $$s^2 = \frac{1}{M} \sum_{i=1}^{M} R_i^2$$
- **当前波动率估计：EWMA和GARCH（特例）** 
- **EWMA模型的推导与应用**
	- **时间轴设定**：以当前时间为0，往前为负数时间轴（如前一天为-1）。
	- **权重设计原则**  
		- 越近的数据权重越高，权重按指数递减：  
		- 第一天权重：$\omega_1$  
		- 第二天权重：$\lambda \omega_1$  
		- 第k天权重：$\lambda^{k-1} \omega_1$
	- **权重归一化**  
		- 所有权重之和为1：  $$\omega_1 (1 + \lambda + \lambda^2 + \cdots + \lambda^{k-1}) = 1$$
		- 解得：    $$  \omega_1 = \frac{1 - \lambda}{1 - \lambda^k} $$
	- **波动率预测公式**
		- $\sigma_n^2 = (1 - \lambda) R_{n-1}^2 + \lambda \sigma_{n-1}^2$
	- **模型特性**  
		- 只用于预测波动率，不能反推历史波动率。  
		- 模型权重递减，反映近期数据对当前波动率影响更大。
- **GARCH模型及其特性**
	- **GARCH模型公式**  
		- $\sigma_n^2 = \omega + \alpha R_{n-1}^2 + \beta \sigma_{n-1}^2$  
		- 其中 $\omega = \gamma V_L$，$\gamma + \alpha + \beta = 1$
	- **长期平均方差项**  
		- $V_L$：长期平均方差，反映市场长期波动趋势。
		- is the long run mean variance rate.
	- **Persistency（持续性）**  
		- $\alpha + \beta$：表示波动率的持续性。  
		- >1：不稳定模型  
		- =1：无均值回归（如EWMA）  
		- <1：具有均值回归特性（多数情况）
		- In GARCH (1,1), the sum of the alpha and beta parameters is called persistence. GARCH (1, 1) is unstable if the persistence > 1. A persistence of 1.0 implies no mean reversion. A persistence of less than 1.0 implies “reversion to the mean”, where a lower persistence implies greater reversion to the mean.
	- **GARCH与EWMA关系**  
	- 当 $\gamma = 0$ 时，GARCH退化为EWMA。
	- EWMA is a special case of GARCH.
- **相关系数与协方差估计**
	- **协方差与相关系数公式**  
		- $$Cov(X,Y) = E[(X - \mu_X)(Y - \mu_Y)]$$  
		- $$\rho = \frac{Cov(X,Y)}{\sigma_X \sigma_Y}$$
	- **使用EWMA估计协方差**  
		- $Cov_n = (1 - \lambda) R_{x,n-1} R_{y,n-1} + \lambda Cov_{n-1}$
	- **使用GARCH估计协方差**  
		- $Cov_n = \omega + \alpha R_{x,n-1} R_{y,n-1} + \beta Cov_{n-1}$
- **多维密度估计  MDE**
	- 不仅考虑时间距离，还考虑情境相似性。  
	- 权重分配基于历史情境与当前情境的相似程度。
	- 更灵活的权重分配方式。
- **隐含波动率（Implied Volatility）**  
	- 当前数据法
	- 通过衍生品市场价格反推得到。  
	- 使用Black-Scholes模型（或其它模型）计算。  
	- **特点**：  
		- **前瞻性**（forward-looking）：反映市场对未来波动的预期。  
		- **模型依赖性强**：若模型假设与现实不符，结果可能失效。  
		- **波动率微笑**（volatility smile）：不同执行价对应的隐含波动率不同。
## 信用风险模型
### 1. 外部和内部
- **信用评级 Rating Scales**
	- **Long-Term Ratings**：Ratings for **bonds** are termed long-term ratings.
	- **Short-Term Ratings**：Ratings for **money market** instruments are termed short-term ratings.
	- **评级分类**  
		- **外部评级**：由专业机构（如标普、惠誉、穆迪）进行。
		- **内部评级**：银行等金融机构自行评估。
	- **评级等级**  
		- **投资级**：BBB级及以上（含BBB-），违约概率低。
		- **投机级**：BBB级以下，违约风险较高。
	- **评级表示方式**  
		- 标普与惠誉使用大写字母（如AAA、BBB）。
		- 穆迪使用首字母大写，其余小写（如Aaa、Baa）。
- **非条件违约概率**  
	- 不考虑其他因素，单指某时间段内违约的概率。
- **条件违约概率**  
	- 在已知前若干年未违约的前提下，某年违约的概率。
	- 公式为：  
    $$
    P(\text{Conditional Default}) = \frac{P(\text{Unconditional Default})}{1 - P(\text{Cumulative Default})}
    $$
	- **举例说明**  
	- 若第5年非条件违约概率为2.45%，前4年累计违约概率为15.87%，则条件违约概率为：
    $$
    \frac{2.45\%}{1 - 15.87\%} = 2.91\%
    $$
- **评级流程 Rating Process**  
	- 初评：基于公司历史、行业、财务数据等。
	- 定期审核：评估是否需升级、降级或维持原评级。
	- **评级费用**  
		- 由被评级公司支付，可能引发利益冲突。
		- 建议由投资者支付费用，以提高客观性。
- **影响因素**
	- **经济周期影响 Impact of Economic Cycle**  
		- **Through the Cycle（TTC）**：
		- 基于历史平均信用水平，长期稳定。长期平均评级，用于监管要求。
		- **Point-in-Time（PIT）**：
		- 基于当前经济状况，短期波动大。短期评级，用于审计要求。
	- **行业影响 Impact of Industry**  
		- 银行违约概率普遍高于非金融企业，即使评级相同。
	- **地理位置影响 Impact of Geography**
	- **评级变化的影响 Impact of Rating Changes**
		- 下调影响大，上调影响不明显。
- **评级转移矩阵**
	- 表示评级在一段时间内的变动概率。
	- **示例解析**  
		- 原始评级为A级，降级至BBB或违约的概率分别为2%和3%。
		- 总降级概率为：$$\frac{2 + 3}{52} = 9.62\%$$
	- **矩阵用途**  
		- 预测信用风险变化。
		- 支持投资决策与风险评估。
- **违约概率替代模型 Alternative to Ratings**
	- **KMV模型**  
		- 基于公司资产价值与负债关系，预测违约概率。
		- 核心公式：  $$P(\text{Default}) = 1 - N(d_2)$$
		- 模型假设：公司违约发生在资产价值低于负债面值时。
	- **模型与BSM模型关系**  
		- 将公司权益视为看涨期权，违约概率对应不行权概率。
- **结构化产品评级问题**
	- **次贷危机反思**  
	  - 评级模型低估违约相关性，导致CDO等产品评级虚高。
	  - 评级依赖历史数据，忽视市场变化。
	- **评级套利**  
	  - 产品设计者利用评级模型漏洞，构造高评级产品。
	- **监管改进**  
	  - 评级机构监管趋严，评级使用范围扩大。
- **内部评级方法** 
	- **评分机制**  
	- 基于多个风险因子加权平均得出评级。
	- **Altman Z-Score模型**  
	- **公式**：  $$Z = 1.2X_1 + 1.4X_2 + 3.3X_3 + 0.6X_4 + 1.0X_5$$
	- **其中**：
		- $X_1$：营运资本/总资产 working capital/total assets
		- $X_2$：留存收益/总资产 retained earnings/total assets
		- $X_3$：税前收益/总资产 earnings before interest and taxes to total assets
		- $X_4$：市值/负债账面价值 market value of equity to book value of total liabilities
		- $X_5$：销售额/总资产 sales/total assets	
	- **判定标准**：
		- $Z > 3$：不违约
		- $Z < 1.8$：高违约风险
		- $1.8 \leq Z \leq 3$：灰色地带
- **Hazard Rate与违约概率**
	- Hazard Rate（风险率）表示单位时间平均违约概率。
	- 服从指数分布：  $$P(\text{No Default in } t) = e^{-\lambda t}$$
	- **累计违约概率**  
		- $P(\text{Default in } t) = 1 - e^{-\lambda t}$
	- **边际违约概率**  
		- $P(\text{Default in } (t_1, t_2)) = e^{-\lambda_1 t_1} - e^{-\lambda_2 t_2}$
### 2. 国家风险
- **国家风险评估**
	- 国家风险主要针对跨国投资者，评估跨国投资时应关注经济、政治、法律等多方面因素。
- **经济周期反应评估** 
	- 评估国家风险时需关注经济周期的反应速度，发展中国家GDP在经济下行时下降幅度远高于发达国家。
- **政治风险** 
	- 政权更迭、政府决策影响行业盈利。Nationalization or expropriation
	- 民主或专制政权的稳定性差异。 Democracies or authoritarian governments
	- 贪污腐败、暴力冲突影响经济稳定性。Corruption Violence.
	- 示例：以色列与巴勒斯坦、印度与巴基斯坦的冲突不利于投资环境。
- **强制征收与国有化问题**  
	- 国有化主要针对自然资源，如土地使用权归国家所有。
- **法律风险** 
	- 法律系统不完善或执行效率低影响投资稳定性。
- **经济比较优势**
	- 经济分散化有助于降低风险，例如泰国依赖旅游业，韩国依赖明星效应。
- **要素环境** 
	- 生产要素（人工、资本）的相对优势。
	- 需求环境（产品和服务需求）。
	- 相关行业竞争力。
	- 公司战略与结构影响企业竞争力。
- **国家违约对本国企业的影响**  
	- 国家违约可能导致国内企业在国际上信誉受损，如俄罗斯主权债券违约对企业的影响。
- **国家风险评分体系**
	- **PRS（Political Risk Service 政治风险评分）**：基于22个维度，分数0-100，越高风险越低。
	- **Euromoney**：基于400位经济学家调研，分数0-100，越高风险越低。
	- **The Economist**：基于外汇、主权债务、银行风险综合评分，分数越高风险越高。
	- **世界银行**：评估腐败、政府执政有效性、政策稳定性、监管质量、法律与会计方式等，得分可为正负。
- **主权信用风险  Sovereign Credit Risk**
	- **外币债券违约风险高于本币债券。Foreign Currency Defaults**
		- 本币可通过印钞偿还债务，外币无铸币权，无法随意印钞。
	- **本币违约原因 Local Currency Defaults**
		- 印钞可能导致通胀，货币贬值。
		- 欧元区国家无独立货币政策。
		- 金本位制下印钞受限（如1971年前）
	- **违约影响 Impact of a Default**
		- 国际市场信誉受损。
		- 融资难度增加。
		- 投资者信心下降。
		- 经济下行、政策不稳定。
		- GDP增长受负面影响。
		- 信用评级下降，影响出口与银行系统。
	- **主权信用评级因素 Sovereign Credit Ratings**
		- 社会福利承诺（现金流流出）。Commitment to social security.
		- 税收收入（现金流流入）。The tax base
		- 政策风险。Political risk.
		- 隐性支持（如行业扶持）。Implicit Guarantees
- **信用利差 Credit Spread**
	- 指主权债务超额利息支付，超过无风险收益率。
	- 表示国家信用风险水平，越高风险越高。
	- 对评级反应迅速，体现风险补偿。
	- 反映新信息速度快于评级。
	- 数据波动性较大，因反应迅速。
	- Credit Supply数据主要来自CDS市场。
### 3. 测量信用风险
- **预期损失（Expected Loss, EL）**
	- 预期损失是银行直接纳入账户的成本，通常通过提高借款利率覆盖。
	- 举例：饭店中偶尔有人白吃白喝，银行将类似损失打入产品价格中，如将利率从4%提升至4.2%。
	- 公式：$EL = PD \times LGD \times EAD = PD\times(1-Recovery Rate)$，其中：
		- $PD$：违约概率
		- $LGD$：损失给付率（Loss Given Default）
		- $EAD$：违约时的风险敞口（Exposure at Default）
- **非预期损失（Unexpected Loss, UL）**
	- 非预期损失是银行用资本金（Capital）覆盖的损失。
	- 举例：突发性重大损失，如地震摧毁餐馆，无法通过日常定价覆盖。
	- 公式：$UL = \text{Worst Case Loss} - EL$
- **最差情境损失（Worst Case Loss）**
	- 指银行所能承受的最大损失上限，超过该上限的损失通常通过保险覆盖。
- **违约概率（PD, Probability of Default）**
	- 违约概率是指借款人违约的可能性。
	- 举例：A借钱给B，B违约不还钱，A面临损失。
	- 违约概率的主体是借款人（B），而非贷款人（A）。
- **风险敞口（Exposure at Default, EAD）**
	- 风险敞口是指借款人违约时，贷款人可能损失的金额。
	- 举例：A借给B 100元，若B违约，A损失100元，即风险敞口为100元。
- **损失给付率（LGD, Loss given Default）**
	- 指借款人违约时，贷款人实际损失的百分比。
	- 举例：若B违约，A未要求抵押品，则LGD为100%；若B提供价值80元的抵押品，则LGD为20%。
- **回收率（Recovery Rate）**
	- 回收率与LGD成反比关系
	- $$\text{Recovery Rate} = \frac{recovery}{exposure}= 1 - \frac{LGD}{exposure}$$
- **贷款组合的信用风险**
	- **单笔贷款的风险分布**
		- 损失分布为二项分布，仅存在违约或不违约两种情况。
		- 均值（预期损失）：$EL = EAD \times LGD \times PD$
		- 方差：$\text{Variance} = EAD^2 \times LGD^2 \times PD \times (1 - PD)$
		- 标准差：$\sigma = EAD \times LGD \times \sqrt{PD \times (1 - PD)}$
	- **贷款组合的风险度量**
	  - 预期损失（ELP）：组合中所有贷款预期损失之和。
	  - 组合方差公式：$$    \sigma_p^2 = \sum_{i=1}^{N} \sigma_i^2 + 2\sum_{i<j} \rho_{ij} \sigma_i \sigma_j$$
	  - 若所有贷款PD、LGD、EAD相同，且相关系数一致，则组合标准差可简化为：$$    \sigma_p = \sqrt{N \cdot \sigma^2 + N(N-1)\rho \cdot \sigma^2}  $$
- **高斯Copula模型** 
	- 用于将非正态分布转换为标准正态分布，并计算联合违约概率。
	- 举例：将两个非正态分布映射为标准正态分布，再通过交集计算联合概率。
	- 缺点：相关系数数量为 $C_N^2$，随贷款数量非线性增长。
- **One Factor模型** 
	- 构建标准正态分布变量 $U_i = a_iF + \sqrt{1 - a_i^2}Z_i$，其中：
	- $F$：共同因子，代表宏观经济变量。标准正态分布。
	- $Z_i$：个体变量，与F不相关。标准呢正态分布。
	- $a_i$：参数，代表F对 $U_i$ 的影响权重。
	- 优点：相关系数由 $a_i \times a_j$ 直接得出，减少计算复杂度。
- **Versa Check模型**
	- 假设所有贷款的 $a_i$ 相同，则相关系数为 $a^2$
	- 用于计算极端情境下的违约率，公式为：$$    \text{Worst Case Default Rate} = N\left( \frac{N^{-1}(PD) + \sqrt{\rho} \cdot N^{-1}(0.999)}{\sqrt{1 - \rho}} \right)    $$
- **信用风险测量方法**
	- **Credit Metrics模型**
	- 利用评级转移矩阵预测信用评级变化，进而评估组合价值变动。
	- 举例：贷款组合当前价值为 $V_0$，若一年后评级下降，组合价值变为 $V_i$，损失为 $V_0 - V_i$

	- **风险配置（Risk Allocation）**
		- 用于决定新增资金应投资于哪个资产以最小化整体风险。
		- 采用欧拉定理和齐次性原理，计算各资产对组合风险的边际贡献。
- **衍生品信用风险测量的挑战**
	- **风险敞口（EAD）计算困难**
		- 衍生品价值随市场波动，敞口难以固定。
		- 举例：看涨期权价值从2元涨至5元，敞口随之上升。
	- **净额结算（Netting）风险**
		- 多笔交易中，若交易对手违约，净额结算失效，敞口上升。
		- 举例：A与B之间有两笔交易，净敞口为200元，若B违约，敞口恢复为800元。
	- **错路风险（Wrong-Way Risk）**
		- 违约概率（PD）与风险敞口（EAD）同时上升，导致预期损失显著增加。
		- 举例：交易对手违约概率上升，同时衍生品价值上升，导致双重风险叠加。
	- **违约相关性估计困难**
		- 多交易对手之间违约相关性难以准确估计，尤其在极端市场条件下。
	- **经济下行对信用风险的影响**
		- 经济下行时，违约概率上升，回收率下降，信用风险加剧。
	- **敞口估计的不确定性**
		- 衍生品敞口受标的资产价格波动影响，难以准确估计。
- **总结**
	- 信用风险主要包括预期损失（EL）与非预期损失（UL）。
	- 预期损失由PD、LGD、EAD三者决定，直接打入产品价格。
	- 非预期损失由资本金覆盖，计算基于风险敞口与违约概率的联合分布。
	- 高斯Copula模型与One Factor模型用于计算多资产违约相关性。
	- 风险配置基于边际风险贡献，决定新增资金最优分配。
	- 衍生品信用风险测量面临敞口估计、净额结算失效、错路风险等挑战。
## 操作风险模型
- **雷曼兄弟破产事件引发的转账失误**
	- **事件背景**：2008年9月15日上午10点，美国第四大投行雷曼兄弟申请破产保护。
	- **错误转账发生**：德国国家发展银行于10点10分通过自动付款系统向雷曼兄弟即将冻结的账户转入3亿欧元，造成无法挽回的损失。
- **操作风险的定义与四大成因**
	- **定义来源**：巴塞尔委员会1995年提出操作风险定义，inadequate or failed internal processes, people, and systems or from external events. 不包含 strategic risks or reputational risks.
	- **四大成因**   
		- **内部流程不完善或失败（Inadequate or failed internal processes）** 
		- 如贷款审批缺少复核流程可能导致违规放贷。
		- **人员因素（People）**   
		- 包括无意失误和主观故意行为，如尼克里森导致巴林银行倒闭。
		- **系统问题（Systems）**
		- 信息系统故障或数据丢失，现代银行设多地数据中心防止单点失效。
		- **外部事件（External events）** 
		- 如自然灾害（地震、海啸）、不可抗力等造成的损失。
- **操作风险包含与排除的风险类型**
	- **包含法律风险（Legal Risk）** 
		- 因法律法规不健全或跨国适用性差异导致的损失归入操作风险。
	- **排除战略风险（Strategy Risk）**
		- 战略决策失误造成的损失难以量化，属商业选择范畴，不在操作风险管理之列。
		- **举例说明** 
		- 2008-2009年为刺激经济，银行大量放贷至光伏、钢铁等行业，后行业违约，属战略风险。
	- **排除声誉风险（Reputational Risk）**
		- 市场负面传闻引发挤兑等影响，虽严重但不属于操作风险。
		- **江苏射阳农商行案例** 
		- 传言行长跑路致储户挤兑，后证实为谣言，属声誉风险。
- **操作风险的七类损失事件**
	- **内部欺诈（Internal Fraud）**
	- 内部人员参与诈骗、盗用资产、篡改账目等行为，如尼克里森伪造利润表。
	- **外部欺诈（External Fraud）** 
	- 第三方实施的诈骗，如短信诈骗要求汇款。
	- **员工雇佣行为与工作场所安全（Employment Practices and Workplace Safety）**
	- 违反劳动法规、健康安全标准引发赔偿。
	- **客户、产品及商业行为相关事件（Clients, Products & Business Practices）**
	- 未能满足客户需求或产品设计缺陷造成损失。
	- **有形资产损失（Damage to Physical Assets）** 
	- 自然灾害或事故导致实体资产损坏。
	- **经营中断或系统出错（Business Disruption and System Failures）** 
	- 软硬件故障、通信中断、设备老化等问题。
	- **执行、交割与交易过程问题（Execution, Delivery and Process Management）**
	- 交易指令未发出、数据输入错误、文件不全、账户越权访问等。
- **新兴的操作风险类别**
	- **网络风险（Cyber Risk）**
	- 数据破坏、金钱盗窃、知识产权窃取、个人信息泄露、钓鱼网站攻击等。
	- **合规风险（Compliance Risk）** 
	- 故意或非故意违反法律法规、内部政策或行业最佳实践，如洗钱、恐怖融资、逃税。
	- **流氓交易员风险（Rogue Trader Risk）** 
	- 个别交易员违规操作造成巨额损失，如尼克里森、大和银行、法国兴业银行案例。
- **操作风险计量的三种方法（巴塞尔协议II）**
	- **基本指标法（Basic Indicator Approach, BIA）**
		- **计算公式**  $$\text{ORC} = \frac{\sum_{i=1}^{N} \text{Gross Income}_i}{N} \times 15\%$$
		- **特点**  
		- 以过去三年正收入平均值乘以固定系数15%，亏损年份不计入分母。
		- **局限性**  
		- 仅反映业务规模，无法体现风险管理水平，适用于小银行。
	- **标准法（Standardized Approach, SA）**
		- **八大业务条线及权重** 
			- 公司金融（Corporate Finance）、交易与销售（Trading and Sales）、支付与清算（Payment and Settlement） → **18%**
			- 商业银行（Commercial Banking）、托管服务（Agency Services） → **15%**
			- 零售银行业务（Retail Banking）、零售经纪（Retail Brokerage）、资产管理（Asset Management） → **12%**
		- **计算步骤** 
			1. 每年各业务条线收入 × 对应权重，求加权总收入；
			2. 将三年结果取平均。
			3. 计算公式 $${Capital}_{\mathrm{SA}}=\frac{\sum_{\mathrm{i=last~three~years}}\max(\sum\mathrm{GI}_{\mathrm{line~1-8}}\times\beta_{\mathrm{line~1-8}},0)}{3}$$
		- **负收入处理规则** 
			- 同一年内负收入可抵消正收入；
			- 若某年加权后总收入 < 0，则按0计算；
			- 分母始终为3，不剔除亏损年份。
	- **高级计量法（Advanced Measurement Approach, AMA）**
		- **前提条件** 
		- 需满足**定性**（如大型复杂机构）与**定量**（至少五年损失数据）标准。
		- **核心原理**  
		- 构建操作风险损失分布，计算99.9%置信水平下的VaR。
		- The **capital** charge for **AMA** is calculated as the bank’s operational value at risk with a **one-year horizon** and a **99.9%** confidence level.
		- **四要素框架**
			- 内部损失数据（Internal Loss Data）
			- 外部损失数据（External Loss Data）
			- 情景分析（Scenario Analysis）
			- 业务环境与内部控制因素（Business Environment and Internal Controls）
		- **频率分布 Frequency Distribution**
			- **常用泊松分布**：$$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$$
		- **严重性分布 Severity Distribution**
			- 常用**对数正态分布**（Log-Normal Distribution），也可用威布尔、伽马等厚尾分布。
		- **整合方法**
			-通常假设频率分布和严重性分布互相独立。
			- 使用**蒙特卡洛模拟（Monte Carlo Simulation）** 将频率与严重性分布结合，生成整体损失分布。
		- **56格模型** 
			- 8个业务条线 × 7类损失事件 = 56个单元格，每个格子独立建模后汇总。
		- **挑战 Data Issues**  
			- 数据稀疏、主观调整多、模型复杂度高、监管难比较。
- **简单标准法（Standardized Measurement Approach, SMA）**
	- **背景与替代对象**
	- 2016年由巴塞尔委员会推出，取代AMA，解决其跨行不可比问题。
	- **两大组成部分**   
		- 业务指标部分（Business Indicator Component, BI）
		- 损失部分（Loss Component）
	- **业务指标构成**
		- ILDC：利息、租赁、股息收入
		- Financial Component：财务相关收入
		- Service Component：服务类收入
	- **优势**  
	- 更稳定、可比性强，按银行规模分五档统一计算，减少模型差异。
- **操作风险资本分配** 
	- **资本分配目标**： 将操作风险资本分配至各业务条线，识别高风险领域并优化管理。
- **幂律法则（Power Law）**
	- 描述极端损失事件的概率分布：$$P(V > X) = K \cdot X^{-\alpha}$$
	- 参数意义：
		- $K$：常数项
		- $\alpha$：尾部厚度参数，$\alpha$ 越小，尾部越厚
		- 应用于拟合高额损失数据，研究长尾分布特征。
- **操作风险管理的五大工具**
	- **归因分析与成本收益权衡（Cost-Benefit Analysis）Causes of Losses**
		- 分析损失根源，评估控制措施的成本与收益，避免过度投入。
	- **教育培训与风险文化建设（Risk Culture）Education**
		- 培养员工风险意识，使其日常行为中主动识别和规避风险。
	- **风险控制与自我评估（RCSA, Risk Control and Self Assessment）**
		- 定期访谈、问卷调查、回顾历史事件，评估现有控制有效性并制定改进计划。
	- **关键风险指标（Key Risk Indicators, KRI）** 
		- 监测预警信号，如：员工离职率过高、交易失败率上升、临时或空缺岗位数量增加。
	- **保险转移与风险管理机制（Insurance）** 
		- 损失分布三段划分：
		- 预期损失 → 用准备金（Reserve）覆盖
		- 非预期损失 → 用资本金覆盖
		- 极端损失 → 通过保险转移
	- **道德风险（Moral Hazard）应对措施** 
		- 设置免赔额、共保条款、保单限制条件。
	- **逆向选择（Adverse Selection）应对措施** 
		- 加强尽职调查，实施差异化保费定价。
## 压力测试

### 1. 压力测试
- 压力测试最经典、最广泛的应用是在**市场风险**领域，其次是**信用风险**。在2008年金融危机后，**流动性风险**的压力测试也变得至关重要。
- **三个基本步骤**：设想极端但可能发生的情景（extreme, unlikely but possible）、确定传导机制、制定未来应急预案。
- **目标**：评估金融机构是否具备足够的资本和流动性资产以应对极端复杂情况。
- **压力测试作为VaR模型的补充**
	- VaR模型仅关注给定置信水平下的分位数损失，无法反映分位数左侧的极端损失程度。
	- **对VaR的有效补充包括**：
		- ES（Expected Shortfall）：计算超过VaR值的极端损失的平均值。
		- Worst Case Analysis（最差情景分析）。
		- 压力测试：研究在极端情景下尾部损失的程度。
- **压力测试与VaR/ES的主要差异**  
	- **分析方法**  
	    - 压力测试采用前瞻性（forward-looking）分析，预测未来三年内公司能否承受极端压力。
	    - VaR 和 ES 采用回顾性（backward-looking）方法，基于历史数据计算极端损失。
	  - **情景选择**  
	    - 压力测试选择少量负面情景（如次贷危机重演、重度压力情景），数量少且均为负面。
	    - VaR 使用历史上真实发生过的数据（如历史模拟法中使用过去500天市场数据），涵盖正负多种情景，数量多。
	  - **时间范围**  
	    - VaR 和 ES 关注短期风险（如巴塞尔协议规定为10天）。
	    - 压力测试关注长期影响，通常需评估未来三年，并按季度计算。
- **压力测试与压力VaR（Stressed VaR）的区别**  
	- **期限长度**  
	    - 压力测试通常覆盖较长时间（如三年）。
	    - 压力VaR 与传统VaR一致，只研究短期（如10天或一个月）。
	- **返回检验（Backtesting）可行性**  
	    - 传统VaR可通过返回检验验证其准确性（如99%置信水平下，250交易日中超限天数应约为2.5天，即2~3天）。
	    - 若实际超限天数达8~10天，则说明VaR设定不准确。
	    - 压力测试因基于未实际发生的假设情景，无法进行返回检验。
- **压力测试的情景选择要点**
	- **时间间隔必须足够长**：覆盖所有相关情景的影响周期。
	- **历史情景的处理方式**：  
	    - 可假设历史变量运行方式不变，将历史事件（如2007–2008年房地产衰退）应用于当前投资组合，即“历史重演”。
	    - 可在历史数据基础上乘以固定倍数（如将波动率扩大至三倍标准差）来构造更严重情景。
	    - 例如：若股票市场历史波动率为10%，可设为30%以评估对投资组合的影响。
	- **关键变量识别**：在传导过程中需识别对投资组合影响重大的风险因子，如：
	    - 股票价格下跌25%
	    - 失业率变化影响信贷水平
	- **专家判断的重要性**：历史不会完全重演，因此专家判断在情景设定中至关重要。可用专家判断替代情景中不合理因素。
- **压力测试结果的应用挑战**  
	- 监管机构要求高管层和董事会审慎评估压力测试结果，但执行难度大。
	- 中国银行业实践中存在两难：
	- 银行不愿呈现过差结果，以免被要求修改模型；
	- 监管机构若设置极端情景却得出乐观结果，也会质疑模型有效性。
- **模型构建的关键要素**  
	- 模型需强调关键变量（key variables），包括：
		- 宏观经济因子（如GDP、CPI变动）
		- 微观企业指标（如资产负债率、利润增长率）
		- 核心变量（core variable）影响附属变量，需明确传导路径。
	- 可采用回归方程建模，但不能仅依赖正常市场历史数据，必须考虑压力情境下的传导机制变化。
	- 需考虑连锁反应（knock-on effect）：多个业务部门同时发生极端事件对整体机构的影响。
- **反向压力测试（Reverse Stress Test）**  
	- 定义：从不利结果出发，逆向推导可能导致该结果的原因。
	- 目的：识别潜在致险因素并预防其发生。
- **美联储的压力测试要求（CCAR）**  
	- 资产规模超过500亿美元的银行必须参与全面资本分析与审查（Comprehensive Capital Analysis and Review, CCAR）。
	- 至少需考虑四种情景：
		- 基础情景（baseline）
		- 温和不利情景（adverse）
		- 严重不利情景（severely adverse）
		- 内部自定义情景（internal scenario）
	- **资本计划提交要求**  
		- 银行须提交资本计划，内容包括所用模型及测试结果。
		- 若测试显示资本不足（capital insufficient），须制定应急方案，如增加外部融资、限制分红
		- 上述措施均可能带来声誉风险，需充分考量。
- **中小银行适用规则（Dodd-Frank Act Stress Testing）**
	- 资产在100亿至500亿美元之间的银行适用《多德-弗兰克法案》压力测试要求。
	- 与CCAR相似，但无需提交资本计划。
### 2. 治理结构
- **董事会职责 The Board**
	- 负有监督责任（oversee），负责公司战略、风险偏好与风险文化。
	- 应规定如何执行压力测试（dictate）。
	- 需评估以下内容：
		- 场景创建过程
		- 模型假设
		- 所用模型本身
		- 具体情景设计
- **高级管理层职责 The Senior Management**
	- 确保由有能力的员工（competent staff）执行压力测试。
	- 定期向董事会报告（regularly report）。
	- 确保机构遵守适当政策与程序，并随经济环境和新风险变化及时调整。
	- 确保压力测试覆盖所有风险敞口和业务条线，使用统一场景，提供全公司视角的风险视图。
	- 考虑测试结果是否需要增加资本或流动性，并仔细分析改进措施。
- **压力测试的政策与流程 Policies and Procedures**
	- 必须建立书面化、有效遵守的政策与流程。
	- 政策应包含：
		- 压力测试的目的说明
		- 实施程序并在全公司范围内执行
		- 明确各角色与职责
		- 定义测试频率（如信用风险每年一次，市场风险每季度一次）
		- 明确场景构建与选择的过程
	- 必须设立独立审查机制。
	- 向第三方（监管机构、审计、评级机构）提供材料时需有清晰文档支持。
	- 明确压力测试结果的使用者。
	- 根据市场变化及时更新方法，允许管理层跟踪结果随时间的变化。
	- 外部获取的模型必须保留相应记录。
- **验证与独立审计 Validation and Independent Review**
	- 审查人员必须保持独立性。
	- 审查内容包括：
		- 压力测试的定性判断方面
		- 模型是否基于有效理论（sound theory）
		- 是否认识到模型的局限性和不确定性
		- 是否持续监控（continually monitor）测试结果
		- 外部模型也必须经过同样严格的检查（不可直接使用）
	- 内部审计部门职责：
		- 确保压力测试方法和结果在全公司体系内一致
		- 协助改进治理、控制与责任机制
		- 提升业务部门能力
		- 向董事会和高管层提供建议
### 3. 巴塞尔委员会的要求
- **基本原则一**：基于内部VaR和ES的市场风险计量，必须伴随严格而全面的压力测试。
- **基本原则二**：计算信用风险资本要求时，也需进行压力测试以评估假设的稳健性与有效性。
- **现存问题**  
	- 一些银行的压力测试未能整合各部门风险敞口。
	- 不同部门专家缺乏合作，难以形成企业级风险观点。
	- 所选场景有时过于温和或持续时间太短。
	- 可能低估不同类型风险产品间的相关性。
	- 特定风险未被充分描述，如：
		- 结构性产品（structural products）
		- 等待证券化的产品
		- 对冲不充分的交易对手信用风险
	- 流动性影响目前也被低估。
- **巴塞尔委员会2017年修订的压力测试原则**  
	- 压力测试框架必须具有明确、正式且可接受的目标。
	- 应具备有效的治理架构（董事会、高管层、验证、审计等职责清晰）。
	- 作为风险管理工具，应为业务决策提供信息支持。
	- 框架必须抓住最重要和最相关的风险，并施加足够严重的压力情景。
	- 组织资源和结构应满足框架目标（资源充足）。
	- 需具备准确、高颗粒度的数据和稳健的IT系统支持。
	- 所用模型和方法应符合场景目标（如敏感性分析模型需匹配目标）。
	- 模型结果和框架应接受挑战并定期检查。
	- 压力测试做法和结果应在不同区域间充分沟通。
